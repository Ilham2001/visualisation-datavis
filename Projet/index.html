L’éducation et l’orientation scolaire et professionnelle : quelle donnée pour quelle efficacité ?


  1. Sujet
    2. Le choix de ce sujet est motivé par notre curiosité de voir la relation que nous pourrions voir entre le processus d’orientation des élèves de terminale entrant dans l’enseignement supérieur et la demande sur le marché de l’emploi.

    Nous avons recueilli les données de parcoursup 2021 et 2022 afin de les analyser par région. L’objectif étant de les superposer avec les données de l'emploi par région disponibles sur le portail de l’INSEE.

  2. Equipe
    3. L’équipe est constitué par trois étudiant.e.s Ilham Lazar, Soraya Haddad, Patrick sanga

  3. Données

    4. Les données utilisées sont issues du portail de data.education.gouv.fr

    2. Données 2021
      3. https://data.enseignementsup-recherche.gouv.fr/api/explore/v2.1/catalog/datasets/fr-esr-parcoursup_2021/exports/csv?lang=en&timezone=Europe%2FBerlin&use_labels=true&csv_separator=%3B
    3. Données du chômage
